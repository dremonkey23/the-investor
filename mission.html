⚠️ Hard Lessons

Margin disaster: Alpaca defaulted to margin. Discovered $27K usage. NEVER enable margin.
X Free tier: Newlines cause 403. All tweets must be single paragraph.
Render ephemeral disk: Without persistent disk, DB wipes on deploy. Added /data disk.
EAS submissions: Builds 6-8 failed silently. "Bundle version must be higher" = Apple already has it.
QA before builds: Always test API endpoints before building. Don't burn EAS credits.
Filter "all selected": When all defaults checked, skip filter entirely or it excludes everything.
SEC company lookup: Use company_tickers.json, NOT search-index (returns people names).
Congressional separate table: Must merge into unified feed via mapCongressional().
